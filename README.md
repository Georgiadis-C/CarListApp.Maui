# CarListApp.Maui 🏎️

A full-stack cross-platform application built with **.NET MAUI** and **ASP.NET Core Minimal API**. This project demonstrates secure user authentication and complex data management.

## 🚀 Features
- **Full Auth System:** JWT-based authentication with Login/Logout functionality.
- **Role-Based UI:** Dynamic menu and Flyout headers based on user roles (Admin/User).
- **Modern API:** High-performance Minimal API with **Scalar/OpenAPI** documentation.
- **Database:** Local and server-side persistence using **SQLite** and **Entity Framework Core**.
- **Security:** Integrated **ASP.NET Core Identity** for robust user management.

## 🛠️ Tech Stack
- **Frontend:** .NET MAUI, XAML, MVVM (CommunityToolkit)
- **Backend:** ASP.NET Core Minimal API, C#
- **Database:** SQLite, EF Core
- **Auth:** JWT (JSON Web Tokens), Identity Framework
